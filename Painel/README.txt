Painel - Empacotamento para produção

Esta pasta contém a aplicação FastAPI empacotada para rodar como executável standalone (painel.exe) no Windows Server.

Arquivos necessários:
- painel.exe
- .env
- certs/
- logs/
- static/

Para instalar como serviço Windows, use NSSM ou sc.exe apontando para painel.exe.
